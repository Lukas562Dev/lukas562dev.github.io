Printers on Arch Linux

ThatOneLukas 2021-03-24 | Tags: linux, arch linux, printer

Here are the steps to use your printer on Arch Linux:

  1. Install cups, system-config-printer and libcups.
sudo pacman -S cups system-config-printer libcups
  1. Enable and start CUPS.
sudo systemctl enable cups
sudo systemctl start cups
  1. Run system-config-printer.
system-config-printer

  1. Press Add in the window that pops up.

  2. Select your printer.

  3. Follow the application's instructions.

  4. You're done!
